Proyecto 2: Web Mercedes-Benz

Descripción del proyecto

Se trata de un proyecto realizado para la asignatura Ingeniería Web con el objetivo de aprender las funciones básicas de HTML5, centrándome en las listas, las tablas y los hipervínculos. Al principio de la página se puede ver la información relevante de la empresa Mercedes-Benz AG junto a un pa de fotos. También hay un enlace que redirige a la página principal de Mercedes-Benz.

En el siguiente punto podemos encontrar las diferentes empresas del grupo Mercedes-Benz con información relevante sobre cada una.

Por último, se encuentra el formulario al realizar para optar uno de los puestos de trabajo ofrecidos.
